# Portfolio Website

Personal portfolio website built with React and TypeScript.

## Tech Stack

- React 18
- TypeScript
- Tailwind CSS
- Vite

## Getting Started

```bash
# Install dependencies
npm install

# Start development server
npm run dev

# Build for production
npm run build
```
